Project Overview

This project is a Deep Learning-based medical image classification system used to detect pneumonia from chest X-ray images. The model analyzes X-ray scans and classifies them as Normal or Pneumonia using Convolutional Neural Networks (CNN).

The main objective of this project is to assist healthcare professionals in early and accurate diagnosis by automating the detection process.

🚀 Features
Chest X-ray image classification
Detects Normal and Pneumonia cases
Image preprocessing and augmentation
CNN-based Deep Learning model
Model accuracy and performance evaluation
Simple and efficient implementation
🛠️ Technologies Used
Python
TensorFlow
Keras
OpenCV
NumPy
Matplotlib
📂 Dataset

The dataset contains chest X-ray images divided into:

Normal
Pneumonia

Images are preprocessed before training to improve model performance.

🧠 Model Workflow
Load and preprocess X-ray images
Resize and normalize images
Train CNN model
Evaluate model accuracy
Predict pneumonia from new X-ray images
📊 Results

The model achieves good accuracy in detecting pneumonia and helps in faster medical diagnosis using AI-based image analysis.
